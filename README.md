## Opis projektu
Jest to aplikacja konsolowa w C# do obsługi uczelnianej wypożyczalni sprzętu.

System pozwala:
- dodawać użytkowników i sprzęt,
- wypożyczać sprzęt,
- zwracać sprzęt,
- naliczać karę za opóźnienie,
- wyświetlać aktywne i przeterminowane wypożyczenia,
- generować krótki raport końcowy.
